What is Stormsound's license?

Stormsound is licensed under the GPL 3.0, so you can use it for any open-source project. We'll be switching to a slightly less restrictive open source license soon, to allow compatibility with permissive-licensed projects.

If you want to use Stormsound in a closed-source project, we'll be offering a paid license upon release.

When will Stormsound be ready?

Stormsound will be in active development soon.

Can I help with development?

We'd love your help! Check out our Developers page for details on how you can get involved.
